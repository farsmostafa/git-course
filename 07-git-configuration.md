# 📝 إعدادات الجيت (Git Configuration) - الدرس السابع

في الدرس ده، هنتعلم إزاي نظبط إعدادات `Git` على جهازنا، زي إننا نعرفه اسمنا والإيميل بتاعنا (عشان لما نعمل `Commit` يظهر للناس مين اللي كتب الكود ده)، وإزاي نتحكم في ألوان الـ `Terminal`، ونشوف كل الإعدادات المتاحة.

## 1. الإعدادات العامة (Global) مقابل المحلية (Local)
* **الإعدادات العامة (`--global`):** لما بتضيف الكلمة دي للأمر، إنت بتقول لـ `Git` طبق الإعداد ده على كل المشاريع (`Repos`) اللي هشتغل عليها على الجهاز ده.
* **الإعدادات المحلية:** لو مكتبتش كلمة `--global`، الإعداد هيتطبق بس على مجلد المشروع اللي إنت واقف جواه حالياً.

## 2. عرض الإعدادات (Listing Configuration)

عشان تشوف كل الإعدادات الحالية المكتوبة على جهازك، بتستخدم الأمر ده.

**القاعدة الأساسية والمثال العملي:**

    git config --list
    أو اختصاراً:
    git config -l

*(ملاحظة: للخروج من قائمة الإعدادات في الـ `Terminal`، اضغط على حرف `q` من الكيبورد).*

## 3. ضبط الاسم والبريد الإلكتروني (User Name & Email)

دي أهم خطوة لازم تعملها أول ما تسطب `Git`، عشان كل حفظ (`Commit`) بتعمله بيتربط بالبيانات دي.

**1. ضبط أو تغيير اسم المستخدم:**
**القاعدة الأساسية (Syntax):**

    git config --global user.name "<Your Name>"

**المثال العملي (Example):**

    git config --global user.name "Fares Moostafa"

**2. ضبط أو تغيير البريد الإلكتروني:**
**القاعدة الأساسية (Syntax):**

    git config --global user.email "<Your Email>"

**المثال العملي (Example):**

    git config --global user.email "fares@example.com"

## 4. قراءة قيمة أو حذف إعداد معين

**1. لقراءة قيمة معينة (مثلاً عايز أعرف الإيميل المتسجل حالياً إيه):**
**القاعدة الأساسية والمثال العملي:**

    git config --global user.email

**2. لحذف إعداد تماماً (Unset):**
**القاعدة الأساسية (Syntax):**

    git config --global --unset <setting_name>

**المثال العملي (Example):**

    git config --global --unset user.name

## 5. تعديل الإعدادات من خلال محرر الأكواد (Editor)

بدل ما تكتب أوامر لكل حاجة، تقدر تفتح ملف الإعدادات نفسه وتعدل فيه كأنه ملف نصي عادي (`Text File`).

**القاعدة الأساسية والمثال العملي:**

    git config --global --edit

لما تفتح الملف ده، هتلاقيه مقسم زي ملفات الإعدادات، وتقدر تضيف جواه إعدادات متقدمة زي تغيير ألوان الـ `Terminal`.

*مثال لتغيير الألوان (يُضاف داخل الملف):*

    [color "status"]
        added = yellow
        changed = red bold
        untracked = magenta bold

*(لو حفظت الملف بالشكل ده، الألوان في شاشة الـ `Terminal` هتتغير لما تكتب أمر `git status` زي ما حددتها بالضبط).*

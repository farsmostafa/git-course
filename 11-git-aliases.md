# 📝 اختصارات الأوامر (Git Aliases) - الدرس الحادي عشر

في الدرس ده، هنتعلم إزاي نسرّع شغلنا ونوفر وقت الكتابة عن طريق إنشاء "أسماء مستعارة" أو اختصارات (Aliases) للأوامر الطويلة اللي بنستخدمها بشكل متكرر كل يوم في `Git`.

## 1. ما هو الـ `Alias`؟

- **المفهوم:** هو إنك تعطي اسم مختصر (حرفين أو تلاتة مثلاً) لأمر طويل أو مركب.
- **الفائدة:** توفير الوقت وتقليل الأخطاء الإملائية عند كتابة الأوامر الطويلة.
- **ملاحظة هامة:** إنشاء اختصار لأمر معين **لا يُلغي** الأمر الأصلي. يعني لو عملت اختصار `st` لأمر `status`، تقدر تستخدم `git st` أو `git status` والاتنين هيشتغلوا معاك عادي جداً.

## 2. كيفية إنشاء اختصار جديد (Alias)

بما إن دي تعتبر "إعدادات" خاصة بيك، فبنستخدم أمر `config` اللي اتعلمناه قبل كده عشان نسجل الاختصار ده على مستوى الجهاز كله (`global`).

### أ. اختصار لأمر من كلمة واحدة

كمثال: عايزين نختصر أمر `status` ليكون `st` فقط، وأمر `branch` ليكون `br`.

**القاعدة الأساسية (Syntax):**

    git config --global alias.<shortcut_name> <command_name>

**الأمثلة العملية (Examples):**

    git config --global alias.st status
    git config --global alias.br branch

_(لتجربة الأوامر بعد إضافتها، اكتب `git st` أو `git br` فقط)._

### ب. اختصار لأمر يتكون من عدة كلمات (يحتوي على مسافات)

لو الأمر اللي عايز تختصره فيه مسافات (زي أمر الاعتماد `commit -m`)، لازم تحط الأمر الأصلي كله بين علامتين تنصيص `" "`.
كمثال: عايزين نختصر أمر `commit -m` ليكون `cm` فقط.

**القاعدة الأساسية (Syntax):**

    git config --global alias.<shortcut_name> "<command_with_spaces>"

**المثال العملي (Example):**

    git config --global alias.cm "commit -m"

_(لتجربة الأمر بعد إضافته، اكتب `git cm "Your commit message"`، وسيتم الحفظ مباشرة)._

## 3. التحقق من الاختصارات أو التعديل عليها

زي ما شفنا في درس الـ `Config`، نقدر نفتح ملف الإعدادات نفسه ونضيف أو نحذف الاختصارات دي يدوياً كأننا بنكتب في ملف نصي عادي.

**لفتح ملف الإعدادات للتعديل:**

    git config --global --edit

لما الملف يفتح، هتلاقي `Git` أنشأ قسم جديد جواه اسمه `[alias]`، وبيكون شكله منظم جداً كالتالي:

    [alias]
        st = status
        br = branch
        cm = commit -m

_(تقدر تضيف اختصارات جديدة مباشرة تحت هذا القسم وتحفظ الملف)._

## 💡 نصيحة ذهبية للبحث:

مش لازم تخترع كل الاختصارات بنفسك وتتعب فيها. تقدر تبحث في جوجل عن جملة `Git Aliases list`، هتلاقي مبرمجين كتير وشركات عاملين قوائم جاهزة لاختصارات عبقرية لأوامر طويلة ومعقدة جداً تقدر تاخدها نسخ وتضيفها عندك توفر عليك سطور كاملة من الكتابة!
